Factory System

Mechanics

  • A hub stores all items, and can be connected to different machines using water channels.
  • Machines connected to the hub can be used for crafting.
  • Can set crafting requests for what machines will make, which are worked on when possible.
  • Machines are built using scraps from recycled structures.

Dynamics

  • There is a limit on how many machines can be built.
  • Player would need to decide the quantity of each machines.
  • The player can specify the number of items needed

Aesthetic

  • The machines should take a relatively small area of the field.
  • Cottage industry style
  • Should be easy to set up and quick to use.
