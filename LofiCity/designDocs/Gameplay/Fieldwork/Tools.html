Tools

The following are the tools available to the player.

Each should:

  • Allow the player to work from a distance.
  • Be satisfying and fast to use.
  • Have multiple use cases.
  • Use the same control scheme.

Water hose

Blast water to where the player is pointing. Stores an amount of liquid. Can be used to water the soil, put out fires, clean objects, move polluted water ...

Terraforming tool

Grabs and stores stone and rock, and can place them somewhere else. Used for terraforming, creating compost, changing soil type ...

Laser tool

Destroy objects into scraps. Can set the power level to allow for less destruction. Used for scrapping machines, trimming plants, harvest animals ...

Gravity gun

Move objects and collect them. hold longer for heavier objects. Used for moving machines, herding, picking up items, fishing ...
