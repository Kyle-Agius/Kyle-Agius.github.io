Cart System

The cart is the portable shop. This can be set up by the player to start selling.

System

  • The player can place the cart down anywhere with enough space.
  • They can walk behind the counter to set up the cart, choosing:
    • Cart variants that attract different customers.
    • What items/dishes customers can buy.
    • The cooking appliances needed.
  • They can call Felix (the chef) to come and help them, if he is available.
  • They can also run the cart alone.
  • Presets can be saved and used to set up the cart faster.
  • Leaving the scene automatically places everything back in the player's inventory.

Actions and Feelings

  • Setting up a cart well would require understanding the customer behaviour in the area.
  • The cart can be set up quickly to allow for spur of the moment decisions.
  • With time, setting up a cart should feel second nature.
  • The cart can be customized, feel home made, and resemble an over packed local shop.
