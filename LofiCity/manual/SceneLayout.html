Scene Layout

At any point in time, two scenes are open: The manager scene and a World Scene.

Manager Scene

This scene stores all GameObjects and Scripts that need to be kept persitant throughout the game, independent of what the current scene is. This includes:

  • The Player
  • The Camera
  • The User Interface
  • Any Persistant Manager Scripts

A GlobalManager Singleton is used so that the current world scene can access certain scripts from the Manager Scene.

World Scene

These scenes store anything that is specific to the currently loaded area.

Most of the elements of these scenes fall under Scene Layers and Object Layers. This allows for having a 3D grid and movement system in 2D.

A CurrentSceneData Singleton is used for easy access of key elements.
