Factory Building Game System

To showcase my current game programming abilities, I built a small slice of a factory building game. This took around three months to complete. Focus was placed on performance, limiting the use of per-frame calculations.

  • Main Features Built
  • Layered Grid System for Building and Player Movement.
  • Item and Inventory System Using Scriptable Objects and Addressable Assets.
  • Factory System with Connected Machines and Crafting Requests.
  • Ecosystem Layers and Cleaning System.
  • Date and Time System.
  • NPC Pathfinding and Schedule System.
  • JSON Based Save System with Save Slots.
  • Auto Generated Documentation.

Thesis Project

As part of my undergraduate thesis project, I used Unity to efficiently mimic real-time style transfer in games for both 2D and 3D scenarios.

For 2D, this was done by loading and swapping stylised sprites. This was extended for 3D by stylising the textures applied to a material. The final result achieved the desired goals for both scenarios.

Flight Sim Dashboard

This project was a University assignment focused on teamwork and professional software development. My group and I built a dashboard for flight sim enthusiasts to use alongside the game, displaying additional information when needed. The ASP .Net Core Framework was used to create the application.

  • Research of the Flight Phases From CLimb to Approach
  • Setup of the Base Application and Page Layout
  • Connected the Application to Various Third Party APIs
  • Connected a Database and Created an Account System
  • Created Many Different Components for the System
